# 미션 - 크리스마스 프로모션

- [ ] 캘린더 대한 정보를 갖는 기능 - Calender

  - [ ] 주말 할인
  - [ ] 특별 할인
- [ ] 이벤트 배지를 부여하는 기능 - EventBadge
